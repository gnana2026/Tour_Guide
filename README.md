# Travel Guide App
This is a simple and interactive **Travel Guide Mobile Application** built using **Flutter**. The app allows users to explore popular tourist destinations around the world with images, ratings, and descriptions.
 
# Features
User Login (basic authentication)
Search places easily
View high-quality images of destinations
Ratings and detailed descriptions for each place
Add places to Favorites
Profile section with saved places
Smooth navigation between screens
Booking button (UI functionality)

# Technologies Used
Flutter (Dart)
Material UI Design
Network Images (for place visuals)

# How it Works
1. User logs into the app
2. Browse or search for travel destinations
3. Click on a place to view detailed information
4. Add places to favorites
5. View saved places in Profile section

# Purpose
This project is designed as a beginner-friendly Flutter application to demonstrate:
Multi-screen navigation
UI design and layout
State management (favorites)
Real-world app structure

💡 This app can be further enhanced by adding backend integration, real booking systems, and authentication services.
